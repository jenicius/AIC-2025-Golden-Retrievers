# Setup tutorial
## 0. Install node.js
- Nếu chưa install node.js thì có thể xem tại: https://nodejs.org/en/download/

## 1. cd tới Frontend
```bash
cd Frontend
```

## 2. Install các packages
```bash
npm install
```
- Ở những lần sau, nếu gặp lỗi khi chạy localhost thì có thể là do có package mới được cập nhật vào. Mọi người nhớ chạy lại dòng lệnh này

## 3. Chạy
```bash
npm run dev
```

- Sau đó, chọn localhost
