# Setup tutorial
## 0. Install node.js
- Nếu chưa install node.js thì có thể xem tại: https://nodejs.org/en/download/

## 1. cd tới Frontend
```bash
cd Frontend
```

## 2. Tải preview images
- Tải các preview image ở Data/Batch1_small: https://drive.google.com/drive/folders/1dUPtycEUNw-i5STDmhRyPoStUPI-lUUM?usp=sharing
- Bỏ Batch1_small vào public. Lưu ý giữ nguyên tên cũ

## 2. Install các packages
```bash
npm install
```
- Ở những lần sau, nếu gặp lỗi khi chạy localhost thì có thể là do có package mới được cập nhật vào. Mọi người nhớ chạy lại dòng lệnh này

## 3. Tạo build
```bash
npm run build
```
- Lưu ý là lúc tạo build sẽ hơi lâu, mọi người đợi tí
- Ở những lần sau, khi code có cập nhật thì mọi người phải build lại 

## 4. Chạy
```bash
npm run preview
```

- Sau đó, chọn localhost
